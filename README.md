# CreativityDump
Output all creativity on your server to files.

## How to use
Open the F1 console with admin and enter the "alldump" command!

管理者で、F1コンソール開いてalldumpってうって。
RUSTサーバ直下に看板とかカセットテープのファイルが作成されるよ。
